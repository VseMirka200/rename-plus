# Ренейм+ (Rename Plus)

Мощную утилиту с графическим интерфейсом, созданную для мгновенного наведения порядка в ваших папках.

Файл `Запуск.pyw` запускает приложение.

---

## Возможности
- **Добавление/удаление текста** – префиксы, суффиксы, удаление части имени файла.
- **Замена текста** – поиск и замена подстрок в имени файла.
- **Нумерация** – автоматическая сквозная нумерация файлов.
- **Использование метаданных** – переименование файлов по дате, тегам и т.п.
- **Изменение формата** – регистр, транслит, замена пробелов в имени файла.
- **Шаблоны** – готовые правила для быстрого переименования файлов.
- **Удаление метаданных** – очистка тегов, комментариев и другой служебной информации из файлов.
- **Конвертация файлов** – преобразование форматов (например, JPG в PNG, MP3 в WAV и т. д.).
