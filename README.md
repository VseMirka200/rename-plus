# Ренейм+ (Rename Plus)

Мощная утилита для переименования файлов с графическим интерфейсом и поддержкой командной строки. Файл `Запуск.pyw` запускает приложение.

---

## Возможности
- **Добавление/удаление текста** - префиксы и суффиксы
- **Замена текста** - простая замена подстрок
- **Регулярные выражения** - мощная замена через regex
- **Изменение регистра** - верхний, нижний, заглавный, title case
- **Нумерация** - автоматическая нумерация файлов
- **Метаданные** - переименование на основе метаданных файлов
- **Шаблоны** - использование шаблонов с переменными

---

### Поддерживаемые форматы

#### Изображения:
`.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`, `.webp`, `.tiff`, `.tif`, `.ico`, `.svg`, `.heic`, `.heif`, `.avif`, `.dng`, `.cr2`, `.nef`, `.raw`

#### Документы:
`.pdf`, `.docx`, `.doc`, `.xlsx`, `.xls`, `.pptx`, `.ppt`, `.txt`, `.rtf`, `.csv`, `.html`, `.htm`, `.odt`, `.ods`, `.odp`

#### Аудио:
`.mp3`, `.wav`, `.flac`, `.aac`, `.ogg`, `.m4a`, `.wma`, `.opus`

#### Видео:
`.mp4`, `.avi`, `.mkv`, `.mov`, `.wmv`, `.flv`, `.webm`, `.m4v`, `.mpg`, `.mpeg`, `.3gp`
